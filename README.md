# Crisp

A very minimal lispy calculator implemented in [zig](https://ziglang.org)

## Operations

- `+` add a sequence of numbers
  - `(+ 1 2 3)`
- `-` subtract two numbers
  - `(- 3 2)`
- `*` multiple a sequence of numbers
  - `(* 3 5 2)`
- `/` divide two numbers
  - `(/ 12 4)`
  - decimals are not supported. meaning: `(/ 2 4) => 0`
